# 01. Продуктовая концепция TuffBox

## Позиционирование

**TuffBox — IDE для Minecraft-сборок.**

Это инструмент для авторов модпаков, серверных сборок и технических игроков, которым нужно создавать, тестировать, отлаживать, версионировать и экспортировать сборки.

## Не лаунчер, а IDE

Обычный лаунчер думает так:

```text
Instance → Launch Minecraft
```

TuffBox должен думать так:

```text
Project → Dependency graph → Profiles → Changes → Test runs → Snapshots → Releases
```

## Целевая аудитория

### 1. Разработчики публичных модпаков

Им нужны:

- контроль зависимостей;
- обновления модов;
- экспорт в `.mrpack` и CurseForge zip;
- changelog;
- тестовые запуски;
- анализ крашей.



### 2. Владельцы Minecraft-серверов

Им нужны:

- client pack;
- server pack;
- проверка client-only/server-only модов;
- инструкции для игроков;
- быстрые хотфиксы;
- тестовые сборки.



### 3. Технические игроки

Им нужны:

- стабильные instances;
- снапшоты;
- rollback;
- оптимизационные профили;
- понятные объяснения крашей.



## Главная ценность

TuffBox превращает сборку из хаотичной папки `.minecraft` в управляемый проект с manifest, lockfile, графом зависимостей, историей изменений и релизами.

## Основные обещания продукта

1. **Никакого хаоса в папках.** Все изменения фиксируются.
2. **Зависимости как код.** Моды и версии описаны в lockfile и графе.
3. **Безопасные изменения.** Перед рисковыми действиями создаётся snapshot.
4. **Понятные краши.** ИИ объясняет лог, но не ломает проект самовольно.
5. **Экспорт одной кнопкой.** `.mrpack`, zip, Prism instance, server pack.
6. **Разрешение ошибок.** Фиксирование действий пользователя для разрешения крашей, ошибок и запоминание, распространение решения другим пользователям с предварительной оценкой эффективности решения.



## Будущая сеть TuffSwarm

Долгосрочная цель — федеративная сеть узлов лаунчера (**TuffSwarm**):

- **Knowledge Torrent** — асинхронный обмен опытом крашей (fingerprint → ActionPlan), без сырых логов;
- **Creation Marketplace** — генерация скриптов/конфигов/сборок на idle GPU за Kudos / premium (PoUW).

Канон для агентов и людей: `[13-tuffswarm-network.md](13-tuffswarm-network.md)`. Сейчас реализован локальный foundation (ActionPlan + Crash KB); P2P runtime ещё нет.

## Продуктовый слоган

> Создавай Minecraft-сборки как проекты: с зависимостями, снапшотами, тестами и понятной отладкой. Быстро чини свой проект с помощью сети общих знаний которые дают ИИ!
